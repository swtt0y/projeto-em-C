
## Sobre o projeto
 
O sistema, desenvolvido na linguagem C, permite o gerenciamento de cadastros de usuários por meio de dois modos de acesso: **administrador** e **usuário**. 

Os cadastros são salvo em arquivos de textos que se mantém atualizado das mudanças.
 
Funcionalidades presentes em cada modo:
 
#### Modo Administrador
- Login com nome e senha protegidos
- Alteração de cadastro de qualquer usuário
- Alteração das próprias credenciais de administrador
- Adição de novos cadastros
#### Modo Usuário
- Verificação de CPF no sistema
- Adição de usuario não cadastrado
- Login com senha após CPF verificado
- Alteração do próprio cadastro
